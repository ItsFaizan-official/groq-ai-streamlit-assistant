## 🚀 About the Project

An AI-powered assistant built using **Streamlit + Groq LLM**, featuring a secure login system and an interactive UI for asking questions about **Python, Data Science, and AI**.

## ⚙️ Installation & Setup

```bash
git clone https://github.com/ItsFaizan-official/groq-ai-streamlit-assistant.git
cd groq-ai-streamlit-assistant
pip install -r requirements.txt
streamlit run app.py
```

---

## 💡 Usage

1. Run the app
2. Login using demo credentials
3. Enter your Groq API key
4. Ask any AI-related question
